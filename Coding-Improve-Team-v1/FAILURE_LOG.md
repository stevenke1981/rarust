# FAILURE_LOG.md

本檔案記錄失敗案例，用於避免 agent 重複犯同樣錯誤。

## Failure Classes

| Class | Meaning |
|---|---|
| Build Failure | 編譯失敗 |
| Test Failure | 測試失敗 |
| Runtime Failure | 執行時失敗 |
| Integration Failure | 模組整合失敗 |
| Requirement Mismatch | 與需求不符 |
| Environment Failure | 環境或依賴問題 |
| Unknown Failure | 尚未定位 |

## Entry Format

```markdown
## YYYY-MM-DD — Failure Title

- Task:
- Class:
- Command:
- Error:
- Attempted Fixes:
- Root Cause:
- Final Resolution:
- Prevention:
```

## Failures

尚無紀錄。
