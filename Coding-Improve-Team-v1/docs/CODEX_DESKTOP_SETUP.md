# Codex Desktop Setup

## Recommended Setup

1. Put this pack inside the project root.
2. Open Codex Desktop in that project directory.
3. Ask Codex to read `AGENTS.md` first.
4. Use the standard task prompt from `usage.md`.

## Project Isolation

Do not run coding tasks from `C:\Users\<name>` unless that is the actual project root.

Each project should own its own:

- `spec.md`
- `plan.md`
- `todos.md`
- `test.md`
- `final.md`
