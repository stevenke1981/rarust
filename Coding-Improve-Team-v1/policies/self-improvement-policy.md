# Self Improvement Policy

## Purpose

讓 agent 從任務中學習，但不要失控改規則。

## Allowed

- Add concise lessons to `lessons.md`.
- Add failure entries to `FAILURE_LOG.md`.
- Suggest improvements to AGENTS/TEAM.

## Not Allowed Without Approval

- Changing core policy.
- Removing safety rules.
- Weakening test gates.
- Auto-updating global settings.
