# Evidence and Testing Policy

## Purpose

完成不能只靠自信，必須靠證據。

## Required Evidence

At least one of:

- Unit test result
- Integration test result
- Build result
- Lint/type-check result
- Manual run evidence
- Screenshot or output log when relevant

## Recording

Write commands and results into `test.md`.

## If Validation Cannot Run

Explain:

- Why it cannot run
- What was checked instead
- What risk remains
