# /self-check

Purpose: 對目前修改做自我檢驗。

## Checklist

1. Did I modify only files related to the task?
2. Did I update `spec.md`, `plan.md`, `todos.md`, `test.md`, and `final.md`?
3. Did I run build/test/lint or explain why not?
4. Did I check for secrets or unsafe commands?
5. Did I add or update tests when behavior changed?
6. Did I avoid broad formatting changes?
7. Can every changed file be justified?
8. Is rollback easy?

## Output

Write a `templates/validation-report.md` style report.
