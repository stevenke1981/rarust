# /auto-code

Purpose: 依照已知需求自動完成 coding 任務。

## Procedure

1. Confirm project root.
2. Read current architecture and relevant files.
3. Convert the request into `spec.md` acceptance criteria.
4. Write `plan.md` with small implementation steps.
5. Write `todos.md` with checkboxes.
6. Implement one step at a time.
7. After each step, run the smallest relevant validation.
8. Update `test.md` with commands and results.
9. Update `final.md`.

## Required Output

- Modified code
- Updated task docs
- Validation evidence
- Known limitations
